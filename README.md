# Real-Time Weather Forecast Application

A responsive weather forecasting web application that fetches live weather data using the OpenWeather API and displays temperature, humidity, weather conditions, and dynamic weather icons based on the entered city.

## Features

1. Search weather by city name
2. Real-time weather data fetching
3. Displays:
  a. Temperature
  b. Humidity
  c. Weather condition
  d. Weather emoji/icons
4. Error handling for invalid city names
5. Responsive and user-friendly UI

## Technologies Used

1. HTML5
2. CSS3
3. JavaScript
4. OpenWeather API

## API Used

- OpenWeather API
```bash
https://openweathermap.org/api
```


## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-app.git
```

---

### 2. Navigate to the Project Directory

```bash
cd foldername
```

---

### 3. Run the Application

You can run the project using either of the following methods:

#### Method 1: Open Directly in Browser
Open the `index.html` file in your preferred web browser.

#### Method 2: Use VS Code Live Server (Recommended)

1. Open the project in VS Code
2. Install the **Live Server** extension
3. Right-click on `index.html`
4. Select **"Open with Live Server"**

The application will start on a local development server.
